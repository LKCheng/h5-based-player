# h5-based-player

基于H5 API 的播放器,可以自己修改按钮实现自定义播放器

- html

```html
   <link href="video.css" rel="stylesheet" type="text/css" />
   <script src="video.js" type="text/javascript"></script>
```

- javascript

```javascript
    U.Control.video({ 
        "width": "1000",    //设置宽度     
        "height":"400",     //设置高度
        "src": "http://fs.1473.cn/3eda2da7-6eee-45fe-b36f-f4809eddc150.mp4",  //视频路径
        "pos": $("body")    //视频的位置
    });
```
